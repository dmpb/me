## :rocket: Resources
JSONPlaceholder comes with a set of 4 common resources:

|   |   |
| :------------ | :------------ |
| /users | 14 users  |
|  /posts | 60 posts   |
|  /comments  | 260 comments  |
| /todos | 20 todos |

## :round_pushpin: Routes
All HTTP methods are supported.

|   |   |
| ------------ | ------------ |
| GET  | /users  |
|  GET | /posts/1/comments  |
|  POST | /users/1/todos  |
|  PUT | /todos/1  |
| DELETE | /todos/1 |
