## Descripción del Proyecto

ToDo App es una aplicación web que permite gestionar tareas mediante una interfaz interactiva.

La aplicación consume la API **JSONPlaceholder**, permitiendo simular operaciones CRUD sobre tareas.

---

## Funcionalidades

| Funcionalidad | Descripción |
|------|------|
| Listado de tareas | Obtención de tareas desde la API |
| Creación de tareas | Permite agregar nuevas tareas |
| Actualización | Marcar tareas como completadas |
| Eliminación | Eliminar tareas |

## Instalación

```shell
git clone https://github.com/dmpb/todo-app-vue.git
npm install
npm run dev
```
