## Descripción del Proyecto

Guess the Number es un juego interactivo en el que el usuario debe adivinar un número generado aleatoriamente entre 1 y 100.

El jugador tiene un máximo de **10 intentos** para descubrir el número correcto.

---

## Reglas

1. El sistema genera un número entre 1 y 100.
2. El jugador ingresa un número.
3. El sistema indica si el número es mayor o menor.
4. El jugador tiene 10 intentos.
