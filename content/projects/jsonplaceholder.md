## Descripción del Proyecto

JSONPlaceholder es una **API REST diseñada para simular un backend real**, permitiendo a desarrolladores probar aplicaciones frontend o integraciones con servicios HTTP.

El proyecto reproduce el comportamiento típico de una API utilizada en aplicaciones web modernas.
